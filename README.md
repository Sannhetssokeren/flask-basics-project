Проект: Основы Flask

01. Описание
Этот проект представляет собой базовое веб-приложение, созданное с использованием микрофреймворка Flask.
- Реализована главная страница с приветственным сообщением.
- Добавлены стили с использованием CSS.

02. Структура файлов
- `app.py`: Основной файл Flask-приложения.
- `templates/`: Директория для HTML-шаблонов.
  - `index.html`: Шаблон главной страницы.
- `static/`: Директория для статических файлов.
  - `style.css`: Файл стилей.
  - `img/`: (Опционально) Директория для изображений.
- `README.md`: Описание проекта.

03. Запуск
Для запуска приложения необходим Python 3.x и установленная библиотека `Flask`.

Установка Flask:
```bash
pip install Flask
```
